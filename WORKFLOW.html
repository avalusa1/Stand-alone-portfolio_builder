🚀 Portfolio Automation Workflow

Interactive visualization of the complete CI/CD pipeline

💡 Click on any step below to see detailed information
1
💻 You Edit portfolio.config.json
Update your portfolio information locally
What you do:
  • Edit personal information (name, email, title)
  • Add/update projects and experience
  • Select avatar (gender + index 1-4)
  • Optionally add custom photo path
File location: portfolio.config.json
↓
2
📤 Git Push to Main Branch
Commit and push your changes to GitHub
Commands:
  • git add portfolio.config.json
  • git commit -m "Update portfolio"
  • git push origin main
Result: Triggers GitHub Actions workflow automatically
↓
3
✅ GitHub Actions Triggered
Workflow starts on ubuntu-latest runner (free)
Automatically:
  • Checks out your repository code
  • Sets up Node.js 18 environment
  • Installs all npm dependencies
  • Ready for build pipeline
Duration: ~30 seconds
↓
4
🔍 Validate Configuration
Checks JSON schema before building
Validates:
  • Required fields present (firstName, email, etc.)
  • Data types correct (string, number, array)
  • Email format valid
  • avatarIndex between 1-4
  • gender is "male" or "female"
If INVALID: Build stops with error message If VALID: Continues to next step
↓
5
🔧 Generate Portfolio Data
Converts JSON config to TypeScript code
Process:
  • Reads validated portfolio.config.json
  • Determines avatar path:
    • If customPhotoPath: uses /public/avatars/custom-avatar.png
    • Else: uses /avatars/{gender}/avatar-{index}.svg
  • Generates src/data/portfolioData.ts
  • TypeScript export ready for components
↓
6
🎨 Process Custom Avatar (Optional)
Converts custom photo to stylized avatar
Only runs if: customPhotoPath is set in config

Processing steps:
  • Reads custom photo from uploads/
  • Square crops to 200x200px (center-focused)
  • Boosts saturation by 20% (vibrant colors)
  • Increases brightness by 5% (clearer look)
  • Applies subtle blur effect (modern aesthetic)
  • Exports as PNG to /public/avatars/custom-avatar.png
Supported formats: JPG, PNG, WebP, GIF Max size: 5MB
↓
7
🏗️ Build Project
Compiles React + TypeScript to optimized dist/
Build steps:
  • TypeScript compiler (tsc) compiles React components
  • Vite bundler optimizes code & assets
  • Tree-shaking removes unused code
  • CSS minification
  • JavaScript minification
  • Generates source maps for debugging
Output: /dist folder with optimized files If BUILD FAILS: Deployment stops, check error logs
↓
8
📦 Upload & Deploy
Pushes built files to GitHub Pages
Deployment:
  • Takes /dist folder (compiled portfolio)
  • Uploads to gh-pages branch
  • GitHub Pages serves files live
  • Your portfolio is now accessible!
URL: https://your-username.github.io/repository-name/
↓
✨
🎉 Portfolio Live!
Your portfolio is now accessible worldwide
What happened:
  • ✅ Config validated
  • ✅ Avatar generated or processed
  • ✅ Project compiled
  • ✅ Deployed to GitHub Pages
